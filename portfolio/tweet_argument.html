A Twitter argument for open source

About this piece

This was an assignment for my ARTX 3600 - Writing for New Media course. I was challenged to write a complete persuasive argument within 10 tweets or less. I chose to write about something that I am passionate about, open source software. I will leave the argument for the tweet, but open source software and licensing has been a big part of my journey to learn code and computers. It was hard pruning my writing down over and over again to make it continually more concise, but when I finally fit it within 8 tweets, not 10 (excluding image credits), I found that I had a very persuasive argument. I was able to take off all the fluff, and get right to the heart of the matter.

The original Tweet thread was lost, this screenshot is all that remains. Most of the argument is intact, but all links are unusable.
