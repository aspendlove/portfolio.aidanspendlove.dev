Easydl CLI

About this piece

Easydl CLI is a utility for downloading videos from YouTube and many other sources through the command line. It uses youtube-dl as a backend, and builds many features on top, such as a redesigned interface and customizable presets for downloading content.
Easydl is one of my first programming projects that I ever worked on. My sophmore year of high school, I used youtube-dl often, and I kept feeling that I could make it easier for my use case. I didn't like having to remember long comand arguments, so I made it ask questions. Youtube-dl's playlist settings confused me, so I integrated that into my program too. I then kept going, adding new features, tailoring it to how I use youtube-dl.

Working on Easydl was the first time that I ever published my work on GitHub, and as I iterated on the original code I kept a log of my changes in GitHub, writing detailed commits and release notes.

It runs on linux and is perfect for downloading both video and audio. It is open source and on Github.
